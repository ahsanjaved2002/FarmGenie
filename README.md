🌾 FarmGenie

FarmGenie is an AI-powered full-stack web platform that unifies land bidding, equipment rental, and an agricultural marketplace into a single digital ecosystem. The platform is designed to simplify agricultural operations for farmers, landowners, and agri-businesses through secure, intelligent, and user-friendly digital solutions.

🚀 Features

🛒 Agricultural Marketplace – Buy and sell farming products with smart price insights

🏞️ Land Bidding System – Post land listings and participate in transparent bidding

🚜 Equipment Rental Module – List and rent farming equipment with availability management

🤖 Bilingual AI Chatbot – English & Urdu support for user assistance

💳 Secure Payments – Stripe integration for safe transactions

📊 User Dashboard – Track activity, listings, and transactions

🔐 Authentication & Authorization – Secure login with Firebase Authentication

☁️ Cloud Deployment – Hosted on Vercel with Supabase backend

🎯 Project Objective

FarmGenie aims to bridge the gap between traditional agricultural practices and modern digital technologies by providing an integrated platform that enhances transparency, efficiency, and accessibility in agricultural transactions.
